# chemical-kg

A GitHub Pages site for a chemical operations knowledge graph focused on desulfurization and denitrification scenarios.

## Live Site

- https://xzx34.github.io/chemical-kg/

## Current Scope

This repository currently focuses on the first milestone:

- build a publishable GitHub Pages site skeleton
- present systems, equipment, operations, faults, and evidence documents
- keep the knowledge graph lightweight and visually clear
- prepare for later graph enrichment and QA generation

## Site Structure

- `/` home portal
- `/graph/` graph visualization page
- `/method/` evidence and methodology page
- `assets/data/demo_payload.json` starter graph payload

## Next Steps

- enrich the graph with better normalized entities
- add stronger evidence chains and source excerpts
- connect the graph to LLM-powered question answering
- collect high-quality QA pairs for SFT
